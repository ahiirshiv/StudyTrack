How to Use StudyNest

Everything you need to know to get the most out of your study companion.

📤

Upload a PDF

Drag and drop or click the upload zone on the Home page. Supports all standard PDFs up to 50MB. Files are recognized by name — re-uploading the same file merges with existing data.

✨

AI Topic Extraction

After upload, StudyNest sends your PDF to Google Gemini AI, which reads the content and extracts chapter/topic names automatically. You can also re-scan anytime from the Dashboard.

📋

Topic Management

In the Dashboard, check off topics as you cover them, mark critical ones as important with ⭐, add your own topics manually, or remove irrelevant ones.

📄

PDF Viewer + Notes

Open any PDF in the viewer. On the right, you have a Notion-style rich text editor — format notes with bold, headings, lists, and more. Notes auto-save per file.

📊

Study Insights

The Insights page shows your overall progress, streaks, important-topic counts, and per-file breakdowns with visual charts.

📅

Calendar Planner

Click any date on the Calendar to add study notes. Days with events show a green dot. Use it to plan sessions and track what you studied each day.

👤

Guest vs Google Account

Guest users get 5 file slots and their data is saved locally in the browser. Sign in with Google to unlock 20 slots. Your Guest ID is shown in the top bar.

🔑

API Key Setup

Open this HTML file in a text editor and find GEMINI_API_KEY near the top of the <script> section. Replace the placeholder with your Gemini API key.

Frequently Asked Questions

Is my data saved when I close the browser? ▼
Yes! All topic data, notes, and calendar events are saved in your browser's localStorage and IndexedDB. They persist between sessions. However, clearing browser data or using private/incognito mode will erase them.
Why is AI topic extraction not working? ▼
Make sure you've set your Gemini API key in the source code. Look for GEMINI_API_KEY = 'YOUR_GEMINI_API_KEY' and replace it. Also ensure the PDF is text-based (not a scanned image).
Can I upload the same PDF again? ▼
Yes! StudyNest identifies files by name. If you upload a file with the same name as an existing one, it will load the existing data (topics, notes) instead of creating a duplicate. This lets you reload a PDF to view it again without losing progress.
What's the difference between Guest and Google sign-in? ▼
Both store data locally on your device. The difference is: Guest accounts get 5 file slots, while Google-authenticated accounts get 20 slots. Your Guest ID is randomly generated per device — use Google sign-in for a consistent identity across devices (though cloud sync is not currently available).
How do I set up Google sign-in? ▼
Create a project in Google Cloud Console, enable the Google Identity API, create an OAuth 2.0 credential, and paste your Client ID into GOOGLE_CLIENT_ID in the source code. Then add your domain to the authorized origins.
